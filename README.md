# Agrotavo - Sistema de Gestão de Physalis

Sistema completo de gestão para produção de Physalis em Ibiúna/SP.

## Acesso

Acesse o sistema em: (https://gustavogodinho80-code.github.io/agrotavo/)

## Funcionalidades

- 📋 Caderno de Campo
- 💰 Registro de Custos
- 📄 Notas Fiscais de Venda
- 📦 Controle de Estoque
- ⚙️ Backup e Importação de Dados

## Como usar

1. Acesse o link acima
2. Os dados são salvos automaticamente no seu navegador
3. Use "Exportar Backup" para fazer cópias de segurança
4. Use "Importar Backup" para restaurar dados
